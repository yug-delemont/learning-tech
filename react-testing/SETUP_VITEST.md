# SETUP_VITEST

## setup for vitest in your js app

`npm i -D vitest` to install vitest

### add test script into your package.json

`
 "scripts": {
    "test": "vitest",
    "test:ui": "vitest --ui"
  },
`

`npm t` OR `npm test` to run the test

`npm run test:ui` run test using vitest ui
